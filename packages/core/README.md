# Core Package

## Overview
The `aau-ais-core` package acts as the foundational layer of the **AAU AIS Lakehouse** architecture. It encapsulates foundational data models, shared utilities, and core service interfaces that downstream modules rely on for consistent and reliable data handling.

## Features
- **Data Models**: Standardized, type-safe data models for consistent representation of lakehouse entities.
- **Utilities**: Reusable helper functions for data validation, transformation, and logging.
- **Configuration**: Centralized environment and runtime configuration management.
